## Обзор проекта

Скрипт реализует текстовый пользовательский интерфейс (TUI) для поиска и замены текста в файлах проекта.  

Особенности:

- Интерактивный ввод параметров через терминал.
- Поддержка включённых/исключённых файлов и папок.
- Использование регулярных выражений для поиска и замены.
- Очистка экрана и удобный интерфейс с подсказками.

Используемые библиотеки:

- [`RatatUI`](https://docs.rs/ratatui/latest/ratatui/) — визуальные компоненты TUI: окна, блоки, параграфы.  
- [`Crossterm`](https://docs.rs/crossterm/latest/crossterm/) — работа с терминалом, события клавиатуры.  
- [`regex`](https://docs.rs/regex/latest/regex/) — поиск и замена текста по регулярным выражениям.

## TUI на RatatUI

TUI предоставляет пользователю **5 интерактивных полей ввода**:

1. **Путь к директории** — пример: `src/data`  
2. **Включённые расширения файлов** — через запятую, пример: `swift,txt`  
3. **Исключаемые директории или файлы** — через запятую, пример: `private,tmp`  
4. **Регулярное выражение поиска** — пример: `Constants\.c(\d+)\.rawValue`  
5. **Регулярное выражение замены** — пример: `Constants.c#$1#.rawValue` (`$1` = первая capture-группа)
